# 安装Vue-Cli前的准备

  1. 安装node依赖，node直接进行傻瓜式安装就行。随同NodeJS一起安装的，还有npm这个包管理工具。

  2. 安装nrm，nrm(npm registry manager )是npm的镜像源管理工具，有时候国外资源太慢，使用这个就可以快速地在 npm 源间切换。

    npm install -g nrm

  3. 使用nrm ls命令，可以查看众多的镜像源。

      npm ---------- https://registry.npmjs.org/
  yarn --------- https://registry.yarnpkg.com/
  tencent ------ https://mirrors.cloud.tencent.com/npm/
  cnpm --------- https://r.cnpmjs.org/
  taobao ------- https://registry.npmmirror.com/
  npmMirror ---- https://skimdb.npmjs.com/registry/

    使用nrm use taobao可以使用淘宝镜像，加快我们安装依赖包的速度。

# 安装Vue-Cli脚手架

安装Vue-Cli脚手架，执行下面的命令，全局安装Vue-Cli：

npm install -g @vue/cli

# 创建demo项目

切换到你想要创建项目的目录，使用脚手架工具创建一个demo项目：

vue create demo

若出现windows使用 Git Bash 上下箭头不生效的情况，可以使用以下方法解决：

  • 在git的安装目录下找到bash.bashrc文件并打开，然后在文件最后一行添加alias vue='winpty vue.cmd'代码就行了。

  • 直接输入数字然后按回车键 例如：要选择第二个就输入2

  • 把创建命令vue create demo修改为 winpty vue.cmd create demo

  • 实在不行，就将Git Bash换成cmd。
