# win 10安装RabbitMQ

# 安装前的准备

由于RabbitMQ是基于Erlang的，在安装RabbitMQ之前，我们需要先安装Erlang。在这里，我们下载Erlang 22.0和RabbitMQ Server 3.7.18。

# win10安装Erlang 22.0

下载Erlang 22.0，双击下载的Erlang文件按照步骤安装Erlang22.0。然后配置Erlang环境变量ERLANG_HOME。

并将%ERLANG_HOME%\bin加入到Path路径中。

接下来，在cmd中测试一下Erlang是否安装成功。

# win10安装RabbitMQ Server

和安装RabbitMQ Server类似，双击RabbitMQ安装文件按照步骤完成RabbitMQ Server的安装。安装完后，我们需要配置环境变量RABBITMQ_SERVER。

并将%RABBITMQ_SERVER%\sbin加入到Path路径中。

检查Path环境变量中的值，可以看到新添加的两个值。

# 启动RabbitMQ web管理页面

成功安装RabbitMQ以后，我们切换当前目录到D:\RabbitMQ Server\rabbitmq_server-3.7.18\sbin输入命令启动RabbitMQ web管理页面。

执行命令，我们成功启动RabbitMQ web管理页面，然后我们选择RabbitMQ Service start命令启动RabbitMQ。

接着，我们访问http://localhost:15672/打开RabbitMQ管理界面。

管理页面初始账号为guest，密码为guest。当我们登录web管理页面后，可以看到RabbitMQ的相关信息。
