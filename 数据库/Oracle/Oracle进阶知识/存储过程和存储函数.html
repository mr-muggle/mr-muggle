# 存储过程和存储函数的基本概念

存储过程和存储函数是指存储在数据库中供所有用户调用的子程序。

  • 相同点：完成特定的功能的程序。
  • 不同点：存储函数可以通过一个return字句返回一个函数的值。

# 存储过程

# 创建或替换存储过程

在Oracle数据库中，使用CREATE PROCEDURE创建一个存储过程，其基本语法为：

create [or replace] procedure 存储过程名(参数列表)
as 
PL/SQL语句;

as相当于declare，在as之后紧跟着的是说明部分。

注意，存储过程只能创建或替换，不能修改。

现在，我们创建一个无参的存储过程。

create or replace procedure sayhelloworld
as
begin
 dbms_output.put_line('Hello World!');
end;

# 两种调用存储过程的方式

调用存储过程有两种方式：

  • 使用exec调用存储过程，例如exec sayhelloworld;
  • 编写PL/SQL调用存储过程

第一种调用方式

set serveroutput on;
exec sayhelloworld();

第二种调用方式

set serveroutput on;
begin
  sayhelloworld();
  sayhelloworld();
end;

# 创建带参数的存储过程

前面我们创建的存储过程是不带参数的，现在，我们创建一个带参数的存储过程。需要注意的是，若一个存储过程带参数，必须使用in或out指明参数是属于输入参数还是输出参数。

create or replace procedure raisesalary(eno in number)
as
 psal emp.sal%type;
begin
 -- 得到员工涨薪水前的工资
 select sal into psal from emp where empno = eno;
 -- 给员工涨100元
 update emp set sal = sal+100 where empno = eno;
 dbms_output.put_line('涨前的工资:'||psal '涨后的工资:'||(psal+100));
end;

一般情况下，我们不在存储过程或存储函数中使用commit或rollback。因为存储过程和存储函数可能被其它程序多次调用。

# 存储函数

# 存储函数的定义

存储函数和存储过程比较类似，不同的是，它必须指定一个return字句，用于返回函数值。其语法如下：

create [ or replace ] function(参数列表) return 函数值类型
as 
plsql子程序体;

接下来，我们来查询一下某个员工的年收入：

create or replace function queryempincome(eno in number) return number
as
 -- 定义变量保存员工的薪水和奖金
 psal emp.sal%type;
 pcomm emp.comm%type;
begin
  select sal,comm into psal,pcomm from emp where empno = eno;
  -- 直接返回年收入，有些员工没有奖金，若一个表达式中含义空值，则整个表达式的结果为空，因此需要对奖金使用nvl函数
  return psal*12+nvl(pcomm,0);
end;

# 调用存储函数

编写完存储函数以后，我们就可以调用存储函数了。

declare
 eno number;
 v_return number;
begin
 eno := null;
 -- 赋值
 v_return := queryempincome(
 eno => eno
 );
 dbms_output.put_line('v_return:'||v_return);
 -- 前面的v_return是输出参数，后面的v_return是存储过程的描述，相当于定义的变量；意思是输出的内容为后面v_return变量中的内容
 :v_return:=v_return;
end;

# 存储过程和存储函数的out参数

一般来讲，存储过程和存储函数的区别在于，存储函数可以有一个返回值，而存储过程没有返回值。存储过程和存储函数都可以指定一个或多个输出参数。我们可以利用out参数，在过程和函数中实现返回多个值。

总而言之：

  • 存储过程和存储函数都可以有out参数
  • 存储过程和存储函数都可以有多个out参数
  • 存储过程可以通过out参数来实现返回值

如果只有一个返回值，用存储函数比较好，如果存在多个返回值，则使用存储过程比较好。

接下来，我们查询一下某个员工的姓名、月薪和职位。由于返回多个值，这里优先使用存储过程。存储过程无法使用return指定返回值，则使用out参数来实现。

create or replace procedure queryempinform(eno in number,pename out varchar2,psal out number,pjob out varchar2) 
as
begin
 --  得到员工的姓名、月薪和职位
 select ename,sal,empjob into pename,psal,pempjob from emp where empno = eno;
end;

# 在out参数中使用使用光标

至此，我们已经学会了存储过程和存储函数的基本用法，若我们有这样一个需求，需要查询某一个部门所有员工的信息。此时可能输出的信息可能会很多，我们需要定义多个输出参数，这样是相当不方便的。因此，我们可以考虑使用光标。

为了使用光标，我们需要定义一个包结构，包括包头和包体。包头用于声明光标和存储过程，包体完成具体的实现。以下是包头的定义：

create or replace package mypackage as
-- 使用type关键字定义一个类型
 type empcursor is ref cursor;
 procedure queryEmpList(dno in number,empList out empcursor);
end mypackage;

定义好包头之后，我们完成包体的实现。

create or replace package body mypackage as
procedure queryEmpList(dno in number,empList out empcursor) as

begin
-- 打开光标
open empList for select * from emp where deptno = dno;
end queryEmpList;
end mypackage;
